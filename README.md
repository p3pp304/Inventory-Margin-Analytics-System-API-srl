# 📦 Warehouse Management System - API Srl

A digital warehouse management system developed to optimize inbound/outbound logistics flows and monitor profit margins in real-time.

---

## 🚀 Project Overview
This repository contains the analytical tools created for the operational management of **API Srl**. The system allows for tracking every warehouse movement, automatically calculating revenues, and visualizing data through interactive dashboards.

### 📂 Repository Structure
* **`screenshots/`**: Visual documentation of the system's core features.
* **`MAGAZZINO API 2024- MODIFICABILE.xlsx`**: Excel file with integrated calculation logic.
* **`Gestione_Magazzino_Api_srl.url`**: Direct shortcut to the cloud view version on Google Sheets.

---

## 📊 Core Features

### 📥 Inbound Stock Management
Purchase monitoring with detailed tracking of:
* Quantities and unit costs of inbound flow of all products.
* Automatic cost calculation and profit.

![Inbound Stock](screenshots/Inbound%20Stock.png)

### 📤 Outbound Stock Management
Sales tracking and outbound flow analysis:
* Total revenue calculation based on selling price.
* Cost monitoring associated with sold goods.
* Instant verification of the operating margin for total outbound flow.

![Outbound Stock](screenshots/Outbound%20Stock.png)

### 📈 Inventory Analytics & Profit Margin
Summary dashboard for management control:
* Real-time overview of current stock levels.
* History of Inbound and Outbound flows for each product

![Inventory Analytics](screenshots/Inventory%20Analytics_Profit%20Margin%20Overview.png)
